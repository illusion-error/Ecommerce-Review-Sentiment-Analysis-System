## 自动化测试记录

命令：

```powershell
docker compose run --rm --no-deps -e DATABASE_BACKEND=sqlite backend python -m pytest -q
```

结果：

```text
5 passed, 7 warnings in 2.07s
```

警告来源为测试库和 Redis 客户端的弃用提醒，不影响本阶段功能验收。
