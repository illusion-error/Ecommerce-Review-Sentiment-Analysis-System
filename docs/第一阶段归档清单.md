## 第一阶段验收项

1. Docker Desktop 已启动。
2. `docker compose up -d --build` 可以启动服务。
3. `GET /api/health` 返回正常。
4. 后端可以连接 MySQL。
5. 后端可以连接 Redis 或使用缓存降级。
6. 单条分析接口可用。
7. 批量分析接口可用。
8. 历史查询、统计、导出接口可用。
9. 文档和报告已归档。
