# 测试用例清单

## 第一阶段范围

本清单用于第一阶段基础可运行版验收，覆盖 Docker 部署、后端接口、数据库、缓存、前端页面访问和结果导出。

| 编号 | 模块 | 测试内容 | 操作 | 预期结果 |
| --- | --- | --- | --- | --- |
| TC-D-01 | Docker | Compose 配置校验 | 执行 `docker compose config` | 配置无语法错误 |
| TC-D-02 | Docker | 一键启动服务 | 执行 `docker compose up -d --build` | backend、frontend、mysql、redis 正常启动 |
| TC-D-03 | Docker | 容器状态检查 | 执行 `docker compose ps` | MySQL 为 healthy，其他服务为 Up |
| TC-D-04 | 后端 | 健康检查 | 请求 `GET /api/health` | 返回 `status=ok` |
| TC-D-05 | 后端 | 单条文本分析 | 请求 `POST /api/sentiment/single` | 返回情感标签、置信度和记录 ID |
| TC-D-06 | 缓存 | 重复文本缓存 | 连续两次请求相同文本 | 第二次响应 `cached=true` |
| TC-D-07 | 后端 | 批量文件分析 | 上传包含 `content` 字段的 CSV | 返回任务 ID 和统计结果 |
| TC-D-08 | 后端 | 批量任务查询 | 请求 `GET /api/tasks/{task_id}` | 返回任务状态和成功数量 |
| TC-D-09 | 后端 | 历史记录查询 | 请求 `GET /api/history` | 返回分页历史记录 |
| TC-D-10 | 后端 | 统计汇总 | 请求 `GET /api/statistics/summary` | 返回正负向统计和缓存统计 |
| TC-D-11 | 后端 | 结果导出 | 请求 `GET /api/export/{task_id}` | 可下载 CSV 或 Excel 结果 |
| TC-D-12 | 前端 | 页面访问 | 访问 `http://127.0.0.1:5173` | Vue 前端页面正常打开 |
| TC-D-13 | 前端 | 前后端联通 | 在前端发起分析请求 | 前端能展示后端返回结果 |

