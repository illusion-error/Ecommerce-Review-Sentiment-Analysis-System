# 后端接口文档

## 1. 基础信息

- 默认地址：`http://127.0.0.1:8000`
- 框架：FastAPI
- 返回格式：

```json
{
  "success": true,
  "message": "ok",
  "data": {},
  "code": 0
}
```

## 2. 健康检查

```http
GET /api/health
```

返回后端版本、数据库模式和缓存状态。

## 3. 单条评论情感分析

```http
POST /api/sentiment/single
Content-Type: application/json
```

请求：

```json
{
  "text": "物流很快，质量也不错，下次还会买",
  "product_id": "phone",
  "comment_time": "2026-07-07"
}
```

返回：

```json
{
  "success": true,
  "message": "分析成功",
  "data": {
    "record_id": 1,
    "text": "物流很快，质量也不错，下次还会买",
    "clean_text": "物流很快质量也不错下次还会买",
    "label": 1,
    "sentiment": "positive",
    "confidence": 0.79,
    "strength": 7.9,
    "cached": false,
    "product_id": "phone"
  },
  "code": 0
}
```

## 4. 批量评论分析

```http
POST /api/sentiment/batch
Content-Type: multipart/form-data
```

字段：

- `file`：CSV/Excel 文件。
- `product_id`：可选。

文件文本列优先识别：`content`、`comment`、`text`、`raw_text`、`评论`、`评价`、`评论内容`。如果都不存在，默认使用第一列。

返回：

```json
{
  "success": true,
  "message": "批量分析完成",
  "data": {
    "task_id": "batch_xxx",
    "total": 2,
    "success_count": 2,
    "failed_count": 0,
    "positive_count": 1,
    "negative_count": 1,
    "avg_strength": 4.8,
    "results": []
  },
  "code": 0
}
```

## 5. 批量任务状态

```http
GET /api/tasks/{task_id}
```

返回任务状态、总行数、成功行数、失败行数、正负向数量和平均强度。

## 6. 历史查询

```http
GET /api/history?page=1&page_size=20&sentiment=positive&product_id=phone&start_time=2026-07-01&end_time=2026-07-09
```

参数：

- `page`：页码，从 1 开始。
- `page_size`：每页数量，1-100。
- `sentiment`：可选，`positive` 或 `negative`。
- `product_id`：可选。
- `start_time`：可选，开始日期，建议格式 `YYYY-MM-DD`。
- `end_time`：可选，结束日期，建议格式 `YYYY-MM-DD`。

## 7. 统计汇总

```http
GET /api/statistics/summary
GET /api/statistics/summary?product_id=phone
```

返回：

- `total`
- `positive_count`
- `negative_count`
- `positive_ratio`
- `negative_ratio`
- `avg_strength`
- `intensity_distribution`
- `cache`

`intensity_distribution` 顺序对应 `0-2`、`2-4`、`4-6`、`6-8`、`8-10` 五个情感强度区间。

## 8. 结果导出

```http
GET /api/export/{task_id}?file_type=csv
GET /api/export/{task_id}?file_type=xlsx
```

用于下载批量任务的分析结果。
